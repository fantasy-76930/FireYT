Install Fantasy Tune

把網站放到主畫面，像 App 一樣打開

安裝後會出現 Fantasy Tune 圖示，不必每次找網址；播放 YouTube 內容時仍需連接網路。

  1. 01 用 Safari 開啟本站

    在 iPhone 或 iPad 的 Safari 打開 Fantasy Tune。

  2. 02 點「分享」

    點方框向上箭頭；若只看到「⋯」，先點它再選「分享」。

  3. 03 加入主畫面

    選「加入主畫面」，開啟「以網頁 App 打開」，最後點「新增」。

  1. 01 用 Chrome 開啟本站

    在 Android 手機的 Chrome 打開 Fantasy Tune。

  2. 02 點右上角「⋮」

    打開 Chrome 選單，找到「加到主畫面」。

  3. 03 選「安裝」

    點「加到主畫面」後選「安裝」，再依畫面確認。

  1. 01 使用 Chrome 或 Edge

    在電腦瀏覽器開啟 Fantasy Tune。

  2. 02 先按「安裝 App」

    若瀏覽器支援直接安裝，本站按鈕會立即顯示確認視窗。

  3. 03 找不到視窗時

    Chrome 從「⋮ → 投放、儲存及分享」安裝；Edge 從「⋯ → 更多工具 → 應用程式」安裝。
