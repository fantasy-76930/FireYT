Fantasy Tune 隱私權政策

Fantasy Tune 是非 YouTube 官方服務，主要整理公開 YouTube 音樂與聲景連結，點擊播放時會前往 YouTube 或 YouTube Music。

最後更新：2026 年 7 月 11 日

本站會處理哪些資料

  • Fantasy Tune 不提供會員登入，不要求姓名、Email、電話或付款資料。
  • 搜尋框只會把你輸入的關鍵字組成 YouTube 搜尋連結，不會送到 Fantasy Tune 自有伺服器保存。
  • 為避免連續播放相同歌曲，本站會在你的瀏覽器本機儲存已播影片 ID、批次日期與輪替位置。這些資料不會傳送到 Fantasy Tune，也不會用來分析或推測你的曲風偏好；清除本站瀏覽資料即可重設。
  • 安裝成 PWA 時，瀏覽器可能快取網站檔案，讓首頁載入更快或可離線顯示部分內容。
  • 分享按鈕會使用瀏覽器內建分享功能；若裝置不支援，會改為複製本站網址到剪貼簿。

YouTube API Services

Fantasy Tune 的每日熱門歌單會使用 YouTube Data API v3 取得公開影片標題、頻道名稱、影片 ID 與縮圖等資料。 音樂資料與縮圖來源為 YouTube / YouTube Data API Services；Fantasy Tune 不儲存、不重新上傳、不內嵌播放 YouTube 影音內容。

首頁顯示歌單時，瀏覽器會向 YouTube 縮圖網域載入公開影片縮圖；療癒直播分類則使用本站製作的原創視覺圖， 不使用特定 YouTube 影片縮圖作為搜尋卡片代表圖。

使用 Fantasy Tune 時，也可能受到 YouTube 服務條款 與 Google 隱私權政策 影響。

到訪計數

Fantasy Tune 使用第三方到訪計數服務顯示今日到訪與總到訪次數。該服務可能接收基本連線資訊，例如 IP 位址、瀏覽器資訊、來源頁與請求時間，用於產生統計數字。

到訪計數圖片會在頁面開啟時自動向第三方服務請求；如果瀏覽器或外掛阻擋該請求，網站主要功能仍可正常使用。

外部連結

點擊 YouTube、YouTube Music、Google API 文件或其他外部網站後，資料處理會依各外部網站的服務條款與隱私權政策為準。 Fantasy Tune 無法控制外部網站如何收集或使用資料。

內容與聯絡

若你認為某個公開來源、縮圖或連結不適合出現在 Fantasy Tune，可以透過 GitHub Issues 告知，我們會檢查並移除不合適項目。

Fantasy Tune 非 YouTube 官方服務。所有影片、音樂、縮圖與相關權利屬原平台、上傳者與權利人所有。
